# SHOT (moon-shot)

SHOT is the knowledge engine, API connector, and RAG provider for Agentic workflows connecting to the local WiKi (Open Knowledge Format Markdown files).

## Features (Roadmap)
- **Dense Vector Search (RAG)**: Chunking local Markdown documents and retrieving using semantic embeddings.
- **Sparse Full-Text Keyword Search**: Powered by MiniSearch for rapid keyword recall.
- **Spaces Map & Dependency Graph**: Resolving connections between pages using relative markdown links as graph edges.
- **Agent API Services**: JSON endpoints for keyword/semantic queries, context ring retrieval, statistics, planet map layouts, and summarizations.
